Pymoo: Optimasi Multiobjektif dalam Python

Pymoo adalah sebuah framework Python open-source yang komprehensif untuk memformulasikan dan menyelesaikan masalah optimasi multiobjektif (Multi-Objective Optimization / MOO). Framework ini menyediakan algoritme canggih, metrik kinerja, metode pengambilan keputusan, serta visualisasi Pareto front.

ℹ️

Catatan: Dokumentasi ini dikompilasi dalam Bahasa Indonesia sebagai panduan ringkas dan modul pembelajaran mandiri agar lebih mudah dipahami oleh mahasiswa dan peneliti lokal.

Fitur Utama Pymoo

🧬 Algoritme Canggih

Dilengkapi algoritme evolusioner state-of-the-art untuk single-objective, multi-objective, dan many-objective optimization (seperti NSGA-II, NSGA-III, MOEA/D, dan CMA-ES).

🧩 Formulasi Modular

Membuat definisi masalah optimasi kustom menjadi sangat mudah. Anda dapat dengan cepat mengatur variabel bertipe kontinu, biner, diskret, permutasi, atau campuran.

🎯 Pengambilan Keputusan (MCDM)

Menyediakan modul Multi-Criteria Decision Making untuk membantu menyaring solusi optimal dari Pareto-optimal set menggunakan Pseudo-weights atau Compromise Programming.

📊 Visualisasi Kaya

Alat visualisasi bawaan untuk ruang objektif berdimensi tinggi menggunakan Scatter Plot, PCP (Parallel Coordinates), Heatmap, Radviz, dan Petal Diagram.

Panduan Instalasi

Pymoo tersedia di indeks paket Python (PyPI) dan dapat diinstal dengan perintah berikut:

# Instalasi versi stabil melalui pip
pip install -U pymoo
💡

Kompilasi C: Secara default, perintah di atas akan mencoba mengompilasi beberapa modul komparasi krusial ke dalam kode biner C untuk meningkatkan kecepatan komputasi. Jika kompilasi gagal, pymoo akan otomatis memasang versi Python murni tanpa menghentikan instalasi.

Untuk menginstal pymoo menggunakan Conda/Anaconda:

conda install -c conda-forge pymoo
