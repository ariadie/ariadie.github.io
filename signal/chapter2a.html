📚 Ikhtisar Bab 2 (Bagian Konvolusi)

Bab 1 telah menetapkan kosakata sinyal (signal) dan sistem (system) serta berargumen bahwa sistem Linear Time-Invariant (LTI) / linier invariant-waktu (LTI) merupakan idealisasi yang paling berguna. Bagian ini (Bab 2a) membuatnya konkret untuk operasi konvolusi (convolution): setiap sistem LTI dijelaskan secara lengkap oleh satu fungsi, yaitu impulse response (respons impuls) h(t) atau h[n], dan hubungan input–output diberikan oleh y = x ∗ h.

Inilah inti konseptual dari buku ini. Setelah convolution dipahami, transformasi Fourier, Laplace, dan z-transform (Bab 3–9) menjadi alat yang alami untuk mengubah convolution menjadi perkalian. (Eigenfunction dan sistem persamaan beda/diferensial dibahas di Bab 2b.)

hImpulse Response

Output dari sistem LTI ketika input-nya adalah impuls satuan δ(t) / δ[n]. Fungsi ini mengkarakterisasi sistem secara lengkap.

∗Convolution

Operasi y = x ∗ h yang menghasilkan output dengan menjumlahkan salinan impulse response yang telah diberi bobot dan digeser.

⚙️Properti

Komutatif/asosiatif/distributif, serta uji causality & stability langsung dari h.

Tujuan pembelajaran (S1/S2): Di akhir bagian ini Anda seharusnya mampu (i) menuliskan convolution sum/integral untuk sistem LTI apa pun, (ii) menentukan causality dan stability langsung dari h, dan (iii) memanfaatkan properti komutatif/asosiatif/distributif.

🔢 Sistem LTI Discrete-Time: Convolution Sum

2.1 Properti Sifting dari δ[n]

Setiap sinyal discrete-time dapat dituliskan sebagai superposisi impuls-impuls yang digeser:

\[ x[n] = \sum_{k=-\infty}^{\infty} x[k]\,\delta[n-k] \](2.1)

Karena sistem bersifat linear, respons terhadap suatu jumlah adalah jumlah dari respons-responsnya; karena bersifat time-invariant (invariant-waktu), respons terhadap δ[n−k] adalah h[n−k]. Memasukkan (2.1) ke dalam sistem menghasilkan:

\[ y[n] = \sum_{k=-\infty}^{\infty} x[k]\,h[n-k] = (x * h)[n] \](2.2)

Inilah convolution sum (jumlah konvolusi). Secara ekuivalen, dengan perubahan variabel,

\[ y[n] = \sum_{k=-\infty}^{\infty} h[k]\,x[n-k] \](2.3)

2.2 Cara Menghitungnya (Flip-and-Slide / balik-dan-geser)

Resep praktis untuk y[n] = x[n] ∗ h[n]:

  • Flip (balik) h[k] untuk memperoleh h[−k].
  • Shift (geser) sebesar n: bentuk h[n−k].
  • Kalikan secara titik-demi-titik dengan x[k] lalu jumlahkan terhadap k.
Disiplin indeks: Indeks penjumlahan adalah k; n adalah parameter. Kesalahan umum adalah menjumlahkan terhadap n. Jaga n tetap saat Anda menggeser.
Contoh mini terselesaikan: Misalkan x[n] = u[n] (unit step) dan h[n] = δ[n] − δ[n−1] (filter beda-pertama). Maka
\[ y[n] = \sum_k u[k](\delta[n-k] - \delta[n-1-k]) = u[n] - u[n-1] = \delta[n] \]
Step diubah kembali menjadi impuls — sistem ini adalah invers dari akumulasi.

🌊 Sistem LTI Continuous-Time: Convolution Integral

2.3 Properti Sifting dari δ(t)

Dalam continuous-time (waktu-kontinyu), impuls berupa delta Dirac, dan setiap sinyal memenuhi:

\[ x(t) = \int_{-\infty}^{\infty} x(\tau)\,\delta(t-\tau)\,d\tau \](2.4)

Berdasarkan linearitas dan time-invariance, respons terhadap x(t) adalah:

\[ y(t) = \int_{-\infty}^{\infty} x(\tau)\,h(t-\tau)\,d\tau = (x * h)(t) \](2.5)

Secara ekuivalen,

\[ y(t) = \int_{-\infty}^{\infty} h(\tau)\,x(t-\tau)\,d\tau \](2.6)

2.4 Membaca Integral

Pikirkan (2.5) sebagai: pada setiap waktu t, akumulasikan kontribusi dari setiap sampel input masa lalu x(τ) yang diberi bobot oleh bagaimana sistem "mengingatnya", yaitu h(t−τ). Inilah sebabnya h disebut impulse response — secara harfiah ia adalah kernel memori sistem.

⚙️ Properti Convolution & Sistem LTI

2.5 Properti Aljabar Convolution

\[ \text{Commutative: } x * h = h * x \](2.7)
\[ \text{Associative: } (x * h_1) * h_2 = x * (h_1 * h_2) \](2.8)
\[ \text{Distributive: } x * (h_1 + h_2) = x * h_1 + x * h_2 \](2.9)

Sifat asosiatif berarti kaskade dua sistem LTI juga merupakan sistem LTI dengan impulse response h = h₁ ∗ h₂ (urutan tidak berpengaruh). Sifat distributif berarti kombinasi paralel memiliki impulse response h = h₁ + h₂.

2.6 Memori

Sebuah sistem LTI bersifat tanpa-memori (memoryless) jika dan hanya jika h(t) = 0 untuk semua t ≠ 0, yaitu h(t) = K·δ(t) (sehingga y(t) = K·x(t)). Selain itu, sistem memiliki memori.

2.7 Causality (Kausalitas)

Sistem LTI yang kausal (causal) harus memenuhi:

\[ h(t) = 0\quad \forall\, t < 0 \qquad (\text{CT});\qquad h[n] = 0\quad \forall\, n < 0 \qquad (\text{DT}) \](2.10)

Intuisi: impulse response tidak dapat dimulai sebelum impuls tiba. Sebuah h yang non-kausal (mis. h(t) = e−|t|) baik-baik saja untuk pemrosesan offline, tetapi tidak dapat direalisasikan secara real time.

2.8 Stability (Stabilitas BIBO)

Sebuah sistem LTI bersifat stabil BIBO jika dan hanya jika impulse response-nya absolutely summable / integrable (dapat dijumlahkan/diintegralkan secara absolut):

\[ \sum_{n=-\infty}^{\infty} |h[n]| < \infty \qquad\text{(DT)};\qquad \int_{-\infty}^{\infty} |h(t)|\,dt < \infty \qquad\text{(CT)} \](2.11)
Ini adalah uji stability yang paling sering digunakan di seluruh buku ini. Untuk low-pass RC, h(t) = (1/RC)e−t/RCu(t) terintegral menjadi 1, sehingga stabil. Untuk integrator yang tidak stabil, h(t) = u(t) terintegral menjadi ∞.

2.9 Invertibilitas

Sebuah sistem LTI bersifat invertibel jika dan hanya jika terdapat hinv dengan h ∗ hinv = δ. Dalam domain frekuensi (bab-bab selanjutnya) hal ini menjadi "tidak pernah nol".

✍️ Latihan Terselesaikan

Latihan 1 — Convolution discrete dari barisan berhingga

Hitung y[n] = x[n] ∗ h[n] untuk x[n] = {1, 2} (tak-nol pada n = 0,1) dan h[n] = {3, 1} (tak-nol pada n = 0,1).

Tampilkan solusi
\[ y[0] = 1\cdot3 = 3 \]
\[ y[1] = 1\cdot1 + 2\cdot3 = 7 \]
\[ y[2] = 2\cdot1 = 2 \]

Jadi y[n] = {3, 7, 2}, tak-nol untuk n = 0,1,2. Panjang = (2 + 2 − 1) = 3 ✓.

Latihan 2 — Causality dari h[n]

Apakah sistem dengan h[n] = (0.5)n u[n+1] bersifat causal? Stabil?

Tampilkan solusi

Causality: h[−1] = (0.5)−1·u[0] = 2 ≠ 0, sehingga h[n] tak-nol untuk n < 0 ⇒ non-causal.

Stability: Σ|h[n]| = Σn=−1∞ (0.5)n = 2 + 1 + 0.5 + … = 4 < ∞ ⇒ stabil.

Latihan 3 — Convolution dengan sebuah delta

Buktikan bahwa x(t) ∗ δ(t−t₀) = x(t−t₀).

Tampilkan solusi
\[ (x * \delta(\cdot - t_0))(t) = \int x(\tau)\,\delta(t-\tau-t_0)\,d\tau = x(t - t_0) \]

berdasarkan properti sifting. Jadi mengonvolusi dengan delta yang digeser hanya menunda sinyal — delta adalah elemen identitas dari convolution, sejauh geseran.

📝 Ringkasan Bagian Konvolusi

Poin-Poin Penting

✓
Impulse response: Sebuah sistem LTI dikarakterisasi sepenuhnya oleh h(t) atau h[n].
✓
Convolution: y = x ∗ h melalui jumlah (DT) atau integral (CT). Flip-and-slide adalah cara praktis menghitungnya.
✓
Hukum aljabar: Convolution bersifat komutatif, asosiatif, dan distributif — kaskade maupun paralel tereduksi menjadi sistem tunggal.
✓
Causality & stability: Causal ⇔ h = 0 untuk waktu/indeks negatif; stabil BIBO ⇔ h dapat dijumlahkan/diintegralkan secara absolut.

Bagian ini mengubah konsep abstrak LTI menjadi alat komputasi konkret: convolution. Impulse response adalah sidik jari lengkap sistem. Lanjut ke Bab 2b untuk eigenfunction dan sistem persamaan beda/diferensial.
