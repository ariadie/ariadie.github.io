📚 Ikhtisar Bab 1

Bab pembuka ini membangun konsep dasar pengolahan sinyal (signal processing) dan analisis sistem (system analysis). Bab ini memperkenalkan kosakata esensial, mendefinisikan klasifikasi sinyal (continuous-time / waktu-kontinyu vs discrete-time / waktu-diskrit, analog vs digital), dan menekankan Linear Time-Invariant (LTI) / linier invariant-waktu (LTI) systems sebagai tulang punggung sebagian besar analisis rekayasa.

Sebelum transformasi Fourier, Laplace, dan z-transform dari bab-bab berikutnya dapat diterapkan, kita harus sepakat tentang apa itu signal (sinyal), apa itu system (sistem), dan bagaimana keduanya berinteraksi. Bab ini sengaja bersifat konseptual: tujuannya adalah kelancaran dengan definisi dan intuisi, bukan komputasi berat. Namun ketepatan definisi-definisi ini menentukan apakah matematika pada Bab 2–9 diterapkan dengan benar.

SApa itu Signal (Sinyal)?

Signal (sinyal) adalah sebuah fungsi yang menyampaikan informasi tentang perilaku atau atribut dari suatu fenomena. Secara formal, signal (sinyal) adalah pemetaan dari suatu himpunan domain (biasanya waktu t atau koordinat spasial) ke himpunan nilai (amplitude). Signal dapat berupa continuous-time, discrete-time, atau digital.

SApa itu System (Sistem)?

System (sistem) adalah sebuah operator atau pemetaan T yang mengubah signal masukan (input) x menjadi signal keluaran (output) y = T{x}. System diklasifikasikan berdasarkan sifat seperti linearitas (linearity), invariant-waktu (time-invariance), kausalitas (causality), memori (memory), stabilitas (stability), dan kebalikannya (invertibility).

🎯Mengapa Mempelajari Signals & Systems?

Konsep-konsep ini muncul di mana-mana: komunikasi, kontrol, akustik, optik, rekayasa biomedis, pemrosesan audio/citra, keuangan, dan lainnya. Penguasaan terhadapnya memberikan bahasa yang diperlukan untuk merancang, menganalisis, dan mengoptimalkan system (sistem) dunia nyata.

Tujuan pembelajaran (S1/S2): Pada akhir bab ini Anda seharusnya mampu (i) mengklasifikasikan signal apa pun yang diberikan, (ii) menghitung operasi sinyal dasar, (iii) menentukan apakah suatu system bersifat linier, invariant-waktu, kausal, stabil, dan bermemori, serta (iv) membedakan sinyal energi (energy signal) dari sinyal daya (power signal).

🔑 Motivasi & Definisi Inti

1.1 Signals (Sinyal) sebagai Objek Matematis

Dalam Oppenheim & Willsky, signal (sinyal) diperlakukan sebagai fungsi matematis. Terdapat dua variabel bebas utama:

  • Waktu t (atau n untuk indeks diskrit) — kasus yang paling umum.
  • Koordinat spasial — misalnya, sebuah citra f(x, y) adalah signal dua dimensi.

Signal real-valued memetakan ke ℝ; signal complex-valued memetakan ke ℂ (penting untuk analisis domain-frekuensi).

\[ x(t) : \mathbb{R} \to \mathbb{R},\qquad x[n] : \mathbb{Z} \to \mathbb{R} \](1.1)

1.2 Systems (Sistem) sebagai Operator

System T menerima signal masukan (input) x (sering disebut eksitasi) dan menghasilkan keluaran (output) y (yakni respons):

\[ y(t) = T\{x(t)\},\qquad y[n] = T\{x[n]\} \](1.2)

Kita sering menggambar system sebagai diagram blok: x(t) → [ T ] → y(t). Untuk system dengan banyak masukan atau banyak keluaran (MIMO), operator memetakan antar ruang vektor, namun bab ini berfokus pada system single-input single-output (SISO).

Peringatan notasi: Jangan mencampuradukkan operator system T{·} dengan variabel waktu. Pada continuous time kita tulis x(t); pada discrete time variabel bebasnya adalah indeks bilangan bulat n, bukan instan pencuplikan t—meski t = nTs menghubungkan keduanya.

1.3 Peran Sentral System LTI

Meskipun system nyata jarang sepenuhnya linier atau invariant-waktu, idealisasi LTI sangat kuat karena:

  • Diharakterisasi sepenuhnya oleh impulse response (respons impuls) h(t) atau h[n].
  • Konvolusi (convolution) mendeskripsikan hubungan masukan–keluaran secara tepat.
  • Pada domain frekuensi, konvolusi menjadi perkalian — dasar dari penyaringan (filtering).
  • Fungsi eigen (eigenfunction) system LTI adalah eksponensial kompleks est / ejωt.

Inilah sebabnya seluruh buku teks membangun ke arah pemahaman konvolusi dan transformasi Fourier/Laplace/z: mereka mendiagonalisasi system LTI.

2.2 Analog vs Digital

  • Analog: continuous (kontinyu) dalam waktu dan continuous dalam amplitude.
  • Discrete-time: terdefinisi pada instan diskrit, tapi amplitude masih bernilai riil (continuous).
  • Digital: diskrit dalam waktu dan terkuantisasi dalam amplitude (himpunan level terhingga).
\[ \text{Digital} \subset \text{Discrete-time} \subset \text{All signals} \](2.1)

2.3 Periodic vs Aperiodic

Signal continuous-time bersifat periodik (periodic) dengan periode T > 0 jika:

\[ x(t) = x(t + T)\quad \forall t,\qquad T_0 = \min\{T > 0\} \](2.2)

Untuk discrete-time, syaratnya adalah x[n] = x[n + N] untuk bilangan bulat N. Sebuah subtlety: sinusoid diskrit cos(Ω0 n) hanya periodik jika Ω0/2π merupakan bilangan rasional. (Coba Ω0 = 1 rad/sample — ia tidak pernah berulang tepat.)

2.4 Genap, Ganjil, dan Simetri Konjugat

Signal apa pun dapat didekomposisi menjadi bagian genap (even) dan ganjil (odd):

\[ x_e(t) = \tfrac{1}{2}[x(t) + x(-t)],\qquad x_o(t) = \tfrac{1}{2}[x(t) - x(-t)] \](2.3)
\[ x(t) = x_e(t) + x_o(t) \](2.4)

Untuk signal kompleks, conjugate symmetry x(-t) = x*(t) adalah yang membuat transformasi Fourier bernilai riil—fakta kunci yang digunakan dalam analisis spektrum.

2.5 Singularitas (Singularity Functions)

Oppenheim membangun banyak signal dari beberapa singularity functions (fungsi singularitas) dasar. Yang paling penting:

  • Unit step u(t): u(t) = 1 untuk t ≥ 0, nol untuk t < 0. Versi diskritnya u[n] bernilai 1 untuk n ≥ 0. Ia berfungsi sebagai "saklar" yang menyalakan signal pada suatu waktu.
  • Unit impulse / Dirac delta δ(t): bukan fungsi biasa melainkan distribusi, dengan δ(t) = 0 untuk t ≠ 0 dan ∫ δ(t) dt = 1. Sifat sifting-nya: ∫ x(t)δ(t−t₀)dt = x(t₀). Diskrit: δ[n] = 1 pada n = 0, nol lainnya.
  • Unit ramp r(t) = t·u(t): bertambah linear setelah t = 0.
  • Hubungan turunan: u(t) = ∫−∞t δ(τ)dτ, sehingga du/dt = δ(t); dan dr/dt = u(t).
\[ \delta(t) = \frac{du(t)}{dt},\qquad r(t) = \int_{-\infty}^{t} u(\tau)\,d\tau \](2.5)
Fungsi-fungsi ini adalah "batu bata" tempat impulse response h(t) di Bab 2 dibangun—setiap LTI system dapat dinyatakan melalui kombinasi δ dan turunannya.

🧮 Operasi Sinyal Elementer

Manipulasi-manipulasi ini muncul berulang di seluruh buku. Misalkan x(t) adalah sebuah signal dan t0, a adalah konstanta.

Time Shift (Pergeseran Waktu / Tunda-Maju)

\[ y(t) = x(t - t_0)\quad \text{delays by } t_0 > 0;\quad x(t + t_0)\text{ advances} \](3.1)

Time Scaling (Percepatan / Perlambatan)

\[ y(t) = x(at),\quad |a| > 1 \text{ compresses, } 0 < |a| < 1 \text{ expands} \](3.2)
Urutan penting: x(2t − 3) tidak sama dengan x(2(t − 3)). Selalu faktorkan: x(2t − 3) = x(2(t − 1.5)) berarti kompresi oleh 2 kemudian tunda oleh 1.5. Urutan yang disarankan adalah: geser dulu (sejauh 3), lalu skala.

Time Reversal (Pembalikan Waktu)

\[ y(t) = x(-t) \](3.3)

Operasi Amplitudo

  • Scaling: y(t) = A·x(t)
  • Addition: y(t) = x₁(t) + x₂(t)
  • Multiplication: y(t) = x₁(t)·x₂(t)
Mini-contoh terbimbing: Diberikan x(t) pulsa persegi satuan dari t = 0 ke t = 1, sketsakan x(2t − 1).
Solusi: tulis sebagai x(2(t − 0.5)). Kompres pulsa dengan faktor 2 (lebar kini 0.5) lalu geser kanan sejauh 0.5. Hasil: persegi dari t = 0.5 ke t = 1.0.

⚡ Sinyal Energi dan Daya

Banyak signal fisik (misalnya, sebuah tegangan melintasi resistor 1 Ω) membawa energi. Kita mendefinisikan energi total dan daya rata-rata:

Continuous-Time

\[ E_x = \int_{-\infty}^{\infty} |x(t)|^2\,dt \](4.1)
\[ P_x = \lim_{T\to\infty} \frac{1}{2T}\int_{-T}^{T} |x(t)|^2\,dt \](4.2)

Discrete-Time

\[ E_x = \sum_{n=-\infty}^{\infty} |x[n]|^2,\qquad P_x = \lim_{N\to\infty} \frac{1}{2N+1}\sum_{n=-N}^{N} |x[n]|^2 \](4.3)
KelasKondisiContoh
Energy signal0 < Ex < ∞, Px = 0pulsa tunggal, eksponensial meluruh
Power signal0 < Px < ∞, Ex = ∞sinus periodik, konstanta
Keduanya bukankeduanya tak terhinggaeksponensial tumbuh et
Sebuah signal tidak dapat sekaligus menjadi energy signal dan power signal (kecuali signal nol yang trivial). Signal periodik selalu merupakan power signal.

⚙️ Sifat Sistem & Uji Mereka

5.1 Linearity (Superposisi)

Sebuah system bersifat linier jika dan hanya jika untuk semua masukan x₁, x₂ dan skalar a, b:

\[ T\{a x_1 + b x_2\} = a\,T\{x_1\} + b\,T\{x_2\} \](5.1)
Uji: Periksa respons masukan-nol. Jika x = 0 memberi y ≠ 0 (misal, y = x + 5), system tersebut tidak linier (paling-paling memiliki offset / linier inkremental).

5.2 Time-Invariance (Invariant-Waktu)

Sebuah system invariant-waktu jika pergeseran waktu pada masukan menghasilkan pergeseran waktu yang identik pada keluaran:

\[ y(t) = T\{x(t)\} \;\Longrightarrow\; T\{x(t - t_0)\} = y(t - t_0) \](5.2)
Jebakan umum: y(t) = x(2t) tidak invariant-waktu. Geser masukan: x(2(t − t₀)) = x(2t − 2t₀), tapi y(t − t₀) = x(2t − 2t₀)… tunggu, bandingkan dengan menggeser keluaran asli: keluaran untuk x(t−t₀) adalah x(2t − t₀). Karena x(2t − 2t₀) ≠ x(2t − t₀), ia gagal. Koefisien internal pada t merusak invariant-waktu.

5.3 Causality (Kausalitas)

Sebuah system kausal (causal) jika keluaran pada waktu apa pun hanya bergantung pada masukan saat ini dan masa lalu: y(t₀) ditentukan oleh {x(t) : t ≤ t₀}. System non-kausal (misal, y(t) = x(t + 1)) tidak dapat direalisasikan secara waktu-nyata namun sah untuk pemrosesan offline / citra.

5.4 Memory (Memori)

Tanpa-memori (memoryless) jika y(t₀) hanya bergantung pada x(t₀). Contoh: y = x² (tanpa-memori), y = x(t−1) (bermemori).

5.5 Stability (BIBO) / Stabilitas (BIBO)

Bounded-Input Bounded-Output stabil jika dan hanya jika setiap masukan terbatas menghasilkan keluaran terbatas:

\[ |x(t)| \le M_x \;\forall t \;\Longrightarrow\; |y(t)| \le M_y \;\forall t \](5.3)
Uji: Untuk system LTI, stabilitas BIBO ⇔ impulse response-nya terintegralkan mutlak: ∫|h(t)|dt < ∞ (CT) atau Σ|h[n]| < ∞ (DT).
💡
Mengapa ini penting: System LTI + kausal + stabil adalah standar emas untuk filter yang dapat direalisasikan. Konvolusi (convolution) sum/dan integral serta berbagai transformasi hanya memberi hasil bersih ketika sifat-sifat ini terpenuhi.

✍️ Latihan Terbimbing

Latihan 1 — Periodisitas sinusoid diskrit

Tentukan apakah x[n] = cos(πn/4) periodik, dan jika ya temukan periode fundamentalnya N₀.

Tampilkan solusi

Sinusoid diskrit cos(Ω₀n) periodik jika dan hanya jika Ω₀/2π = m/N rasional. Di sini Ω₀ = π/4, sehingga

\[ \frac{\Omega_0}{2\pi} = \frac{\pi/4}{2\pi} = \frac{1}{8} = \frac{m}{N} \]

Memilih m = 1 memberi N = 8. Jadi x[n] periodik dengan periode fundamental N₀ = 8.

Latihan 2 — Dekomposisi genap / ganjil

Dekomposisi x(t) = e−t (terdefinisi untuk semua t) menjadi komponen genap dan ganjil.

Tampilkan solusi
\[ x_e(t) = \tfrac{1}{2}(e^{-t} + e^{t}) = \cosh t \]
\[ x_o(t) = \tfrac{1}{2}(e^{-t} - e^{t}) = -\sinh t \]

Cek: cosh t − sinh t = e−t ✓.

Latihan 3 — Uji sifat system

Apakah system y(t) = t·x(t) (a) linier, (b) invariant-waktu?

Tampilkan solusi

(a) Linier: T{a x₁ + b x₂} = t(a x₁ + b x₂) = a·t x₁ + b·t x₂ = a y₁ + b y₂. ✓ Linier.

(b) Invariant-waktu: Geser masukan sejauh t₀: T{x(t−t₀)} = t·x(t−t₀). Tapi y(t−t₀) = (t−t₀)·x(t−t₀). Karena t·x(t−t₀) ≠ (t−t₀)·x(t−t₀), ia tidak invariant-waktu. (Faktor eksplisit t membuat parameter system melenceng terhadap waktu.)

Latihan 4 — Energi vs daya

Klasifikasikan x(t) = 3 cos(2π·50·t) sebagai energy signal atau power signal, dan hitung dayanya.

Tampilkan solusi

Ia periodik ⇒ sebuah power signal. Daya rata-rata A cos(ωt) dalam satu periode adalah A²/2.

\[ P_x = \frac{3^2}{2} = 4.5\ \text{(units of power on 1 Ω)} \]
