Perencanaan Jalur Multi-Robot dalam Lingkungan Dinamis Menggunakan Multiobjective Genetic Algorithm

Disertasi Program Doktor — Ariadie Chandra Nugraha
Beranda Abstrak Nomenklatur

Disertasi ini mengusulkan kerangka perencanaan jalur multi-robot berbasis Non-dominated Sorting Genetic Algorithm II (NSGA-II) dengan inisialisasi populasi A*, formulasi per-agen berkendala, dan operator genetika domain-spesifik untuk menghasilkan solusi Pareto yang feasible pada lingkungan statis maupun dinamis.
