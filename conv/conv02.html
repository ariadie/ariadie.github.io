📚 Ikhtisar Bab 2

Bab ini membahas convex sets — blok bangunan fundamental dari convex optimization. Feasible set dari convex optimization problem selalu merupakan convex set. Kita akan mempelajari definisi, contoh-contoh kanonik (hyperplane, halfspace, polyhedra, ball, ellipsoid, cone), operasi yang melestarikan convexity, generalized inequalities via proper cones, separating/supporting hyperplane theorems, dual cones, dan minimum vs minimal elements (Pareto optimality).

⬢Definisi

Convex set: mengandung line segment antara dua titik manapun di dalamnya. Affine set: mengandung seluruh line.

📐Contoh Kanonik

Hyperplane, halfspace, polyhedra, Euclidean ball, ellipsoid, norm balls, positive semidefinite cone.

🔧Operasi Pelestar

Intersection, affine image/preimage, perspective/linear-fractional, Minkowski sum, convex hull.

🔺Cones & Inequalities

Proper cones definisikan generalized inequalities (componentwise, Loewner, second-order). Dual cone untuk duality.

🔢 2.1 Affine dan Convex Sets

Definition (Affine set): \(C \subseteq \mathbb{R}^n\) affine jika \(\forall x_1, x_2 \in C,\; \forall \theta \in \mathbb{R}\): \(\theta x_1 + (1-\theta) x_2 \in C\).

Affine set mengandung seluruh line melalui dua titik manapun di dalamnya. Ekivalen: \(C = x_0 + V\) di mana \(V\) subspace, \(x_0 \in C\). Dimensi affine set = dimensi \(V\).

Definition (Convex set): \(C \subseteq \mathbb{R}^n\) convex jika \(\forall x_1, x_2 \in C,\; \forall \theta \in [0, 1]\): \(\theta x_1 + (1-\theta) x_2 \in C\).

Convex set mengandung line segment antara dua titik manapun. Setiap affine set adalah convex; konversnya tidak selalu benar.

Convex combination: \(x = \theta_1 x_1 + \cdots + \theta_k x_k\) dengan \(\theta_i \ge 0,\; \sum \theta_i = 1\).
Convex hull: \(\text{conv } C = \{ \sum \theta_i x_i \mid x_i \in C,\; \theta_i \ge 0,\; \sum \theta_i = 1 \}\). Convex set terkecil yang mengandung \(C\).
Contoh Convex Hull di \(\mathbb{R}^2\): Convex hull dari 3 titik = segitiga (termasuk interior). Dari 2 titik = line segment.

📐 2.2 Contoh Penting Convex Sets

Hyperplanes dan Halfspaces

Hyperplane: \(\{x \mid a^T x = b\}\) dengan \(a \ne 0\). Affine (dan convex). Dimensi \(n-1\).

Halfspace: \(\{x \mid a^T x \le b\}\) (closed) atau \(\{x \mid a^T x < b\}\) (open). Convex, bukan affine.

Polyhedra dan Polytopes

Polyhedron: Intersection finitely many halfspaces/hyperplanes: \(P = \{x \mid Ax \le b,\; Cx = d\}\). Convex.

Polytope: Bounded polyhedron. Ekivalen: convex hull dari finite set of points.

Balls dan Ellipsoids

Euclidean ball: \(B(x_c, r) = \{x \mid \|x - x_c\|_2 \le r\}\). Center \(x_c\), radius \(r > 0\).

Ellipsoid: \(\mathcal{E} = \{x \mid (x - x_c)^T P^{-1} (x - x_c) \le 1\}\) dengan \(P \in \mathbb{S}^n_{++}\). Alternatif: \(\mathcal{E} = \{x_c + A u \mid \|u\|_2 \le 1\}\) untuk \(A\) nonsingular.

Norm Balls

Untuk norm \(\|\cdot\|\) apa pun, \(\{x \mid \|x\| \le 1\}\) convex (triangle inequality + homogeneity). Unit ball of any norm is convex.

Positive Semidefinite Cone

\(\mathbb{S}^n_+ = \{X \in \mathbb{S}^n \mid X \succeq 0\}\) (symmetric PSD matrices). Convex cone. \(\mathbb{S}^n_{++} = \{X \mid X \succ 0\}\) — convex tapi bukan cone.

Simplex

Probability simplex di \(\mathbb{R}^n\): \(\Delta^n = \{x \mid \mathbf{1}^T x = 1,\; x \succeq 0\} = \text{conv}\{e_1, \dots, e_n\}\).

🔧 2.3 Operasi yang Melestarikan Convexity

Intersection: Intersection sembarang collection convex sets adalah convex. \(S = \bigcap_\alpha S_\alpha\) convex jika setiap \(S_\alpha\) convex.
Affine image: Jika \(S\) convex dan \(f(x) = Ax + b\), maka \(f(S) = \{Ax + b \mid x \in S\}\) convex.
Affine preimage: Jika \(S\) convex, maka \(f^{-1}(S) = \{x \mid Ax + b \in S\}\) convex.
Perspective & linear-fractional: Perspective function \(P(x, t) = x/t\) (domain \(t > 0\)) memetakan convex sets ke convex sets. Linear-fractional \(f(x) = (Ax + b)/(c^T x + d)\) juga melestarikan convexity.
Example — LMI set: \(\{x \mid x_1 A_1 + \cdots + x_m A_m \preceq B\}\) adalah convex (affine preimage of \(\mathbb{S}^n_+\)).

Minkowski Sum

Jika \(S_1, S_2\) convex, sum \(S_1 + S_2 = \{x + y \mid x \in S_1, y \in S_2\}\) convex.

Convex Hull

Convex hull dari sembarang set adalah convex.

🔺 2.4 Generalized Inequalities

Proper cone: \(K \subseteq \mathbb{R}^n\) proper cone jika: (1) convex, (2) closed, (3) solid (nonempty interior), (4) pointed (\(x \in K, -x \in K \Rightarrow x = 0\)).

Proper cone induksi generalized inequality:

  • \(x \preceq_K y \iff y - x \in K\)
  • \(x \prec_K y \iff y - x \in \text{int } K\)

Minimum/Minimal elements: \(x \in S\) minimum jika \(x \preceq_K y\; \forall y \in S\). Minimal jika \(y \preceq_K x, y \in S \Rightarrow y = x\). Minimum \(\Rightarrow\) minimal; konvers holds jika \(S\) totally ordered (e.g., \(K = \mathbb{R}^n_+\)).

✂️ 2.5 Separating dan Supporting Hyperplanes

Separating Hyperplane Theorem: Jika \(C, D \subseteq \mathbb{R}^n\) nonempty disjoint convex sets, \(\exists a \ne 0, b\) s.t. \(a^T x \le b\; \forall x \in C\) dan \(a^T x \ge b\; \forall x \in D\).
Strict separation: Jika \(C\) closed, \(D\) compact, \(C \cap D = \emptyset\), maka \(\exists a \ne 0, b\) dgn \(a^T x < b\; \forall x \in C\), \(a^T x > b\; \forall x \in D\).
Supporting hyperplane to \(C\) at \(x_0 \in \partial C\): \(a^T x \le a^T x_0\; \forall x \in C\) dgn \(a \ne 0\). Exist untuk setiap boundary point convex set.
Supporting Hyperplane Theorem: Untuk nonempty convex \(C\) dan \(x_0 \in \partial C\), \(\exists\) supporting hyperplane to \(C\) at \(x_0\).
Intuisi: Separation theorems = fondasi geometri duality. "Disjoint convex sets can be separated by a hyperplane" — ide yang sama underlie duality gap & optimality conditions (KKT).

🔄 2.6 Dual Cones dan Generalized Inequalities

Dual cone: \(K^* = \{y \mid y^T x \ge 0\; \forall x \in K\}\). Selalu closed convex cone (bahkan jika \(K\) tidak).

Properties:

  • \(K^* = (\text{cl } K)^* = (\text{conv } K)^*\)
  • \(K_1 \subseteq K_2 \Rightarrow K_2^* \subseteq K_1^*\)
  • \(K^{**} = \text{cl}(\text{conv } K)\) (jadi \(K^{**} = K\) iff \(K\) closed convex cone)
  • \(K\) proper \(\iff\) \(K^*\) proper

Dual Generalized Inequality

\(x \preceq_K y \iff \lambda^T x \le \lambda^T y\; \forall \lambda \succeq_{K^*} 0\)

Ini menghubungkan generalized inequalities ke scalar inequalities — krusial untuk duality theory.

Dual cones (self-dual cones):
  • \((\mathbb{R}^n_+)^* = \mathbb{R}^n_+\)
  • \((\mathbb{S}^n_+)^* = \mathbb{S}^n_+\) (di bawah trace inner product)
  • \((Q^n)^* = Q^n\)

📊 2.7 Minimum dan Minimal Elements

Minimum element of \(S\) w.r.t. \(\preceq_K\): \(x \in S\) minimum jika \(x \preceq_K y\; \forall y \in S\).
Minimal element: \(x \in S\) minimal jika \(y \preceq_K x, y \in S \Rightarrow y = x\).
Characterization via dual cone: \(x\) minimum of \(S\) iff \(\lambda^T x = \inf_{z \in S} \lambda^T z\; \forall \lambda \succeq_{K^*} 0\). \(x\) minimal iff \(\exists \lambda \succ_{K^*} 0\) s.t. \(\lambda^T x = \inf_{z \in S} \lambda^T z\).
Pareto optimality: Untuk \(K = \mathbb{R}^m_+\), minimal elements = Pareto optimal points. Dual cone characterization: point Pareto optimal iff minimizes weighted sum \(\lambda^T x\) dengan \(\lambda > 0\) (weighted sum scalarization).

✍️ Latihan Terselesaikan

Latihan 1 — Convex Hull

Temukan convex hull dari set \(C = \{(x, y) \in \mathbb{R}^2 \mid x^2 + y^2 = 1\}\) (unit circle).

Tampilkan solusi

Convex hull = closed unit disk: \(\text{conv } C = \{(x, y) \mid x^2 + y^2 \le 1\}\).

Set \(C\) hanya boundary lingkaran. Convex hull mengisi interiornya karena line segment antara dua titik di boundary melewati interior.

Latihan 2 — Intersection of Halfspaces

Buktikan bahwa polyhedron \(P = \{x \mid Ax \le b,\; Cx = d\}\) adalah convex set.

Tampilkan solusi

Setiap row dari \(Ax \le b\) mendefinisikan halfspace \(\{x \mid a_i^T x \le b_i\}\) yang convex. Setiap equality \(c_j^T x = d_j\) adalah intersection dua halfspaces: \(\{x \mid c_j^T x \le d_j\} \cap \{x \mid -c_j^T x \le -d_j\}\), juga convex. \(P\) adalah intersection finitely many convex sets \(\Rightarrow\) convex (intersection preserves convexity).

Latihan 3 — Separating Hyperplane

Diberikan \(C = \{x \in \mathbb{R}^2 \mid x_1^2 + x_2^2 \le 1\}\) dan \(D = \{x \in \mathbb{R}^2 \mid x_1 \ge 2\}\). Temukan separating hyperplane.

Tampilkan solusi

\(C\) = unit disk centered at origin. \(D\) = halfspace \(x_1 \ge 2\). Distance between sets = 1.

Separating hyperplane: \(x_1 = 1.5\) (i.e., \(a = (1, 0), b = 1.5\)).

Check: \(\forall x \in C, x_1 \le 1 < 1.5\). \(\forall x \in D, x_1 \ge 2 > 1.5\). Strict separation holds karena \(C\) compact, \(D\) closed.

Latihan 4 — Dual Cone

Temukan dual cone dari second-order cone \(Q^n = \{(x, t) \in \mathbb{R}^n \times \mathbb{R} \mid \|x\|_2 \le t\}\).

Tampilkan solusi

\(Q^n\) self-dual: \((Q^n)^* = Q^n\).

Proof sketch: \((y, s) \in (Q^n)^* \iff y^T x + s t \ge 0\; \forall (x, t) \text{ with } \|x\|_2 \le t\).

Ambil \(x = -t \frac{y}{\|y\|_2}\) (untuk \(y \ne 0\)), maka \(y^T x + s t = -t \|y\|_2 + s t = t(s - \|y\|_2) \ge 0\) untuk \(t > 0\) \(\Rightarrow s \ge \|y\|_2\). Jadi \((y, s) \in Q^n\). Konvers juga holds.

Latihan 5 — Pareto Optimal

Diberikan feasible set \(S = \{(x_1, x_2) \mid x_1 + x_2 \ge 1,\; x_1, x_2 \ge 0\}\) dengan cone \(K = \mathbb{R}^2_+\). Temukan minimal elements (Pareto frontier).

Tampilkan solusi

\(S\) adalah region di atas line \(x_1 + x_2 = 1\) di kuadran pertama.

Minimal elements w.r.t \(\mathbb{R}^2_+\): titik-titik yang tidak bisa diimprove satu punya \(y \preceq x\) dengan \(y \in S, y \ne x\).

Pareto frontier = line segment \(x_1 + x_2 = 1,\; x_1, x_2 \ge 0\). Untuk \(x\) di line ini, tidak ada \(y \in S\) dengan \(y_1 \le x_1, y_2 \le x_2\) dan strict inequality untuk salah satunya.

Verification via dual cone: \(\lambda = (\lambda_1, \lambda_2) > 0\). Minimize \(\lambda_1 x_1 + \lambda_2 x_2\) s.t. \(x_1 + x_2 \ge 1, x \ge 0\). Optimum at boundary \(x_1 + x_2 = 1\).
