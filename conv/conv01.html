📚 Ikhtisar Bab 1

Bab ini memperkenalkan mathematical optimization dan mengapa convex optimization adalah kelas masalah yang paling penting dan praktis. Kita akan melihat formulasi standar, kelas-kelas masalah (LP, QP, SOCP, SDP, GP), dan mengapa convex = tractable.

fMathematical Optimization

Minimasi fungsi objektif dengan constraint. Variabel x, objektif f₀(x), constraint fᵢ(x) ≤ 0, hᵢ(x) = 0.

⬢Convex Optimization

Objektif convex, constraint inequality convex, constraint equality affine. Local minimum = global minimum.

⚡Tractable

Interior-point methods solve in polynomial time. Modern solvers (ECOS, MOSEK, Gurobi) handle thousands of variables in seconds.

🔢 1.1 Mathematical Optimization

Sebuah mathematical optimization problem (atau mathematical program) memiliki bentuk standar:

\[ \begin{aligned} \text{minimize} \quad & f_0(x) \\ \text{subject to} \quad & f_i(x) \le 0, \quad i = 1,\dots,m \\ & h_i(x) = 0, \quad i = 1,\dots,p \end{aligned} \]

Di mana:

  • x ∈ ℝⁿ adalah optimization variable (decision variable)
  • f₀ : ℝⁿ → ℝ adalah objective function (cost function)
  • fᵢ : ℝⁿ → ℝ adalah inequality constraint functions
  • hᵢ : ℝⁿ → ℝ adalah equality constraint functions
Feasible point: x yang memenuhi semua constraint (fᵢ(x) ≤ 0, hᵢ(x) = 0) dan x ∈ dom f₀ ∩ ⋂ dom fᵢ ∩ ⋂ dom hᵢ.
Optimal value: \(p^* = \inf \{ f_0(x) \mid f_i(x) \le 0,\; h_i(x) = 0 \}\). Jika feasible set = ∅, maka \(p^* = +\infty\). Jika unbounded below, \(p^* = -\infty\).
Optimal point: \(x^*\) feasible dan \(f_0(x^*) = p^*\).
Locally optimal: ∃ R > 0 sedemikian \(f_0(x^*) = \inf \{ f_0(x) \mid \text{feasible},\; \|x - x^*\| \le R \}\).
Implicit constraints: Domain fungsi (dom f₀, dom fᵢ, dom hᵢ) mendefinisikan constraint implisit. Misalnya \(\log x\) secara implisit menuntut \(x > 0\).

⬢ 1.2 Convex Optimization

Suatu problem disebut convex optimization problem jika:

  • Objektif \(f_0\) adalah convex function
  • Constraint inequality \(f_i\) adalah convex functions
  • Constraint equality \(h_i\) adalah affine functions (bentuk \(a_i^T x = b_i\))
\[ \begin{aligned} \text{minimize} \quad & f_0(x) \\ \text{subject to} \quad & f_i(x) \le 0, \quad i = 1,\dots,m \quad (f_i \text{ convex}) \\ & a_i^T x = b_i, \quad i = 1,\dots,p \quad (\text{affine}) \end{aligned} \]
Sifat Fundamental: Dalam convex optimization, feasible set adalah convex set, dan objektif convex di atas feasible set tersebut. Setiap local minimum adalah global minimum.
Example 1 — Least Squares (Unconstrained Convex):
\[ \text{minimize} \quad \|Ax - b\|_2^2 \]

Objektif quadratic convex. Solusi analitik: \(x^* = A^\dagger b\) (pseudoinverse). Diselesaikan dalam \(O(n^2 m)\) flops.

Example 2 — Linear Programming (LP):
\[ \begin{aligned} \text{minimize} \quad & c^T x \\ \text{subject to} \quad & Ax \le b \end{aligned} \]

Objektif dan constraint affine (convex). Tidak ada solusi analitik umum; diselesaikan interior-point atau simplex dalam polynomial time.

Example 3 — Non-Convex (Combinatorial):
\[ \begin{aligned} \text{minimize} \quad & c^T x \\ \text{subject to} \quad & Ax \le b \\ & x_i \in \{0,1\} \end{aligned} \]

Constraint \(x_i \in \{0,1\}\) membuat feasible set non-convex (himpunan titik diskrit). Masalah ini NP-hard secara umum.

📋 1.3 Kelas-Kelas Masalah Convex Optimization

KelasObjektifConstraintAplikasi
LPLinear \(c^T x\)Linear \(Ax \le b\)Resource allocation, network flow, scheduling
QPQuadratic \(\frac{1}{2}x^T P x + q^T x\)LinearPortfolio optimization, MPC, SVM
QCQPQuadraticQuadraticTrust regions, robust optimization
SOCPLinear\(\|A_i x + b_i\|_2 \le c_i^T x + d_i\)Robust LP, antenna array, localization
SDPLinear \(\text{tr}(CX)\)LMI \(X \succeq 0,\; F(x) \succeq 0\)Combinatorial relax., control, MIMO
GPPosynomialPosynomial ≤ 1Circuit design, geometric problems
Hierarchy: LP ⊂ QP ⊂ QCQP ⊂ SOCP ⊂ SDP. Semua GP dapat diubah menjadi convex via log-change of variables \(y_i = \log x_i\).

Contoh: Portfolio Optimization (QP)

\[ \begin{aligned} \text{minimize} \quad & x^T \Sigma x \\ \text{subject to} \quad & \mathbf{1}^T x = 1, \quad \mu^T x \ge \mu_{\text{target}}, \quad x \succeq 0 \end{aligned} \]

\(\Sigma\) = covariance matrix (PSD), \(\mu\) = expected returns. Minimize risk (variance) untuk target return tertentu.

Contoh: Robust Linear Programming (SOCP)

\[ \begin{aligned} \text{minimize} \quad & c^T x \\ \text{subject to} \quad & \sup_{a_i \in \mathcal{E}_i} (a_i^T x) \le b_i, \quad i=1,\dots,m \end{aligned} \]

Dengan \(\mathcal{E}_i\) elipsoid, constraint ini menjadi SOCP constraint.

💻 1.4 Software untuk Convex Optimization

Praktik modern menggunakan Disciplined Convex Programming (DCP) — framework yang memverifikasi convexity otomatis dan memanggil solver:

🐍CVXPY (Python)

pip install cvxpy — Paling populer untuk Python. Dukung ECOS, OSQP, MOSEK, Gurobi, CLARABEL.

📊CVX (MATLAB)

Sintaks cvx_begin ... cvx_end. Mature, widely used di akademia.

🔬Convex.jl (Julia)

High performance, native Julia. Integrates dengan JuMP.

🔧YALMIP (MATLAB)

Alternative ke CVX, mendukung lebih banyak solver & problem types.

Contoh CVXPY — Least Squares dengan Box Constraints

import cvxpy as cp
import numpy as np

n, m = 100, 50
A = np.random.randn(m, n)
b = np.random.randn(m)

x = cp.Variable(n)
objective = cp.Minimize(cp.sum_squares(A @ x - b))
constraints = [0 <= x, x <= 1]

prob = cp.Problem(objective, constraints)
result = prob.solve()  # Auto-selects ECOS/OSQP

print("Optimal value:", result)
print("x = ", x.value)

Latihan 1 — Mengenali Convex Problem

Mana dari berikut yang merupakan convex optimization problem?

  1. \(\text{minimize } x^3 \text{ subject to } x \ge 0\)
  2. \(\text{minimize } x^2 \text{ subject to } x \ge 0\)
  3. \(\text{minimize } -x^2 \text{ subject to } x \ge 0\)
  4. \(\text{minimize } \|x\|_1 \text{ subject to } Ax = b\)
Tampilkan solusi

(2) dan (4) adalah convex.

  • (1) \(x^3\) bukan convex untuk \(x \ge 0\) (second derivative \(6x \ge 0\) tapi fungsi bukan convex di seluruh domain — wait, actually \(x^3\) is convex for \(x \ge 0\)! But objective should be convex over whole domain... correction: \(x^3\) is convex for \(x \ge 0\) since \(f''(x) = 6x \ge 0\). So (1) is also convex. My mistake — let me recheck.
  • Actually: \(x^3\) has \(f''(x) = 6x\). For \(x \ge 0\), \(f''(x) \ge 0\) so it IS convex on the feasible set. But the standard definition requires convexity on the whole domain... The feasible set is \(x \ge 0\), and on this set \(f(x)=x^3\) is convex.
  • Let me be precise: (1) is convex on the feasible set \(x \ge 0\). (2) is convex everywhere. (3) \(-x^2\) is concave, NOT convex. (4) \(\|x\|_1\) is convex, equality constraint affine — convex.

Jawaban benar: (1), (2), (4) — maaf untuk kebingungan awal, ini ilustrasi penting: convexity diperiksa pada feasible set!

Latihan 2 — Equivalent Transformations

Ubah problem berikut ke standard form LP:

\[ \text{minimize } \|x\|_1 \quad \text{subject to } Ax \le b \]
Tampilkan solusi

Introduce variabel \(t \in \mathbb{R}^n\):

\[ \begin{aligned} \text{minimize} \quad & \mathbf{1}^T t \\ \text{subject to} \quad & -t \preceq x \preceq t \\ & Ax \le b \end{aligned} \]

Ini adalah LP dengan variabel \((x, t)\). Constraint \(-t_i \le x_i \le t_i\) ekuivalen dengan \(t_i \ge |x_i|\). Pada optimum, \(t_i = |x_i|\).

Latihan 3 — Geometric Programming

Mengapa problem berikut bukan convex dalam variabel aslinya, tapi menjadi convex setelah transformasi log?

\[ \begin{aligned} \text{minimize} \quad & x_1^{-1} x_2^{-2} \\ \text{subject to} \quad & x_1^2 x_2 \le 1 \\ & x_1, x_2 > 0 \end{aligned} \]
Tampilkan solusi

Posynomial (sum of monomials with positive coefficients) bukan convex. Tetapi dengan substitusi \(y_1 = \log x_1, y_2 = \log x_2\):

\[ \begin{aligned} \text{minimize} \quad & e^{-y_1 - 2y_2} \\ \text{subject to} \quad & e^{2y_1 + y_2} \le 1 \end{aligned} \]

Atau ambil log dari objektif & constraint (monotonic):

\[ \begin{aligned} \text{minimize} \quad & -y_1 - 2y_2 \\ \text{subject to} \quad & 2y_1 + y_2 \le 0 \end{aligned} \]

Ini adalah LP dalam \(y\)! GP selalu bisa diubah ke convex via log-transform.

Poin-Poin Penting

✓
Mathematical optimization: Minimasi \(f_0(x)\) dengan constraint \(f_i(x) \le 0, h_i(x) = 0\). Optimal value \(p^*\), optimal point \(x^*\).
✓
Convex optimization: \(f_0, f_i\) convex, \(h_i\) affine. Feasible set convex. Local minimum = Global minimum.
✓
Kelas masalah: LP, QP, QCQP, SOCP, SDP, GP. Hierarchy: LP ⊂ QP ⊂ QCQP ⊂ SOCP ⊂ SDP.
✓
Tractability: Interior-point methods solve in polynomial time. Solvers modern (ECOS, MOSEK, Gurobi) sangat efisien.
✓
DCP tools: CVXPY, CVX, Convex.jl memverifikasi convexity otomatis & memanggil solver.
